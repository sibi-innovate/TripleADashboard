Unable to load dashboard libraries

Safari may block external scripts when opening local HTML files. Please try one of these solutions:

Option 1: Open in Google Chrome instead (recommended)
Option 2: In Safari → Settings → Advanced → check "Disable local file restrictions"
Option 3: Run a local server: open Terminal, cd to the folder, then python3 -m http.server 8000 and visit localhost:8000
